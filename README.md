# Quizable

A simple, interactive quiz application built with HTML, CSS, and JavaScript. This application tests users on fundamental web development concepts (HTML, CSS, and JavaScript).

## Features

- **Interactive Questions**: 10 multiple-choice questions focusing on front-end web development.
- **Real-time Feedback**: Immediate visual feedback showing correct and incorrect answers.
- **Progress Tracking**: A dynamic progress bar that updates as you navigate through the quiz.
- **Results Screen**: A comprehensive summary at the end of the quiz showing the total score, percentage, and a personalized message.
- **Responsive Design**: A clean, modern UI that works well on both desktop and mobile devices.
